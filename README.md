# Junior/Middle IT Job Vacancy Bot

Telegram-бот для автоматического сбора вакансий для Junior и Middle IT-специалистов.

## 🎯 Возможности
- 🔍 Парсинг вакансий из API
- 📊 SQLite база данных
- 💬 Публикация в Telegram
- 🔄 Автообновление

## 🚀 Установка
\\\ash
